## Core Features

QIMEM combines enterprise-grade encryption with a complete identity management system.

<CardGroup cols={2}>
  <Card
    title="Deterministic Encryption"
    icon="lock"
    href="/concepts/encryption"
  >
    AES-256-GCM encryption with envelope format for secure key rotation and multi-region support.
  </Card>
  <Card
    title="QAuth Identity Service"
    icon="user-shield"
    href="/concepts/qauth-identity"
  >
    JWT issuance, RBAC, MFA with TOTP, token refresh, and revocation in a single platform.
  </Card>
  <Card
    title="Key Lifecycle Management"
    icon="key"
    href="/concepts/key-rotation"
  >
    Create, rotate, and manage encryption keys with automatic version tracking and backward compatibility.
  </Card>
  <Card
    title="Plugin System"
    icon="puzzle-piece"
    href="/api/plugins/manifests"
  >
    Register and manage runtime plugins with capability-based access control.
  </Card>
</CardGroup>
